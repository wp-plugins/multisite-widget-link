== Description ==
Simply add widgets in order to link another blog in a multisite instance
This plugin needs a network installation.

== Installation ==

1. Upload the folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress network admin
